■ 확인 및 정리
    1. 참조타입에 대한 설명 중 틀린 것은? ④
	① 참조타입은 배열, 열거, 클래스, 인터페이스가 있다.
	② 참조타입 변수의 메모리 생성 위치는 스택이다
	③ 참조 타입에서 ==, != dustkswksms rorcp qjswlfmf qlrygksek
	④ 참조 타입은 null 값으로 초기화할 수 없다

    2. String 타입에 대한 설명으로 틀린것은? ②
	① String은 클래스이므로 참조타입니다
	② String은 타입의 문자열 비교는 ==를 사용해야 한다
	③ 동일한 문자열 리터럴을 저장하는 변수는 동일한 String 객체를 참조한다
	④ new String("문자열")은 문자열이 동일하더라도 다른 String 객체를 생성한다.
